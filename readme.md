# koishi-plugin-chatluna-smash-or-pass-ai

_上传一张图片，让 LLM 根据其吸引力来评判能不能冲_

## 特性

让 LLM 帮你判断这张图片能不能对着冲

## 部署

在 Koishi 插件市场搜索 `chatluna-smash-or-pass-ai`，安装后启用即可。

**插件依赖 ChatLuna，请确保在 ChatLuna 启动后再启用本插件。**

## 使用

命令：

```fuck [image]```
